>pip install mysqlclient // if its not getting install the do following steps (install this at venv level)
-Go to: https://www.lfd.uci.edu/~gohlke/pythonlibs/#mysqlclient 
download wheel as per your version of python and desktop 32bit or 64bit and install virtual level: 
>pip install <your downloaded file name will come here> hit enter then

- After successfully installing above if stil get error then run the following query to your MySQL database
